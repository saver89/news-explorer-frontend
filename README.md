# Проект News

### Обзор
Поиск новостей с возможностью сохранения по ключевым словам для каждого зарегистрированного пользователя.

### Технологии
- CSS, JS, HTML, 
- React, 
- Webpack

### Команды
- npm start - Запуск проекта в режиме разработчик
- npm run deploy - Релиз проекта
